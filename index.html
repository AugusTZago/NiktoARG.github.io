Mis proyectos

Domotica

Este proyecto de maqueta de casa automatizada o domótica, se realizó con arduino a partir de diversos sensores y actuadores con los cuales se consigue controlar las diversas luces, a parte de contar con un módulo RFID 522 utilizado para controlar el acceso a la puerta. Contando también con un sensor de temperatura y humedad, el cual envia los datos a la aplicación, dicha aplicación se utiliza para el control de las luces.
